# Alsality Group

Corporate website for Alsality Group — hospitality, restaurant development, brand operations, and Athar Marketing.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lenis (smooth scroll)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
